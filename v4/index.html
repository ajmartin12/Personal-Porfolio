Snapper

Photo Sharing Application

Github

Heroku

Snapper is a photo uploading website where users can view other peoples photos, share and orangise them into albums.

Built on Ruby Sinatra connecting to a PSQL Database to save data. The front-end used Bootstrap 3 for the responsive grid and CSS animate.

WDI Conference

Sign-Up Conference Website

Github

Heroku

WDI Group Conf, an app that can sign up users to a conference, find events and attend them.

We implemented a system for users to meet up with other users who did not know each other.

The aim was to get the users networking with new people. We build the app on Ruby on Rails with materialize CSS Framework, HandleBars, AJAX, APIs and jQuery .

SmallTalk

Conversation Starter App

Github

Heroku

SmallTalk is a app designed to help users start conversations by asking them questions and displaying facts

eg. “When was your first time overseas?” or “Did you know cats were worshiped by the ancient Egyptians”.

Built on Ruby on Rails with an ActiveRecord PSQL Database with API integration and AJAX calls. Front-end used Materialize CSS Framework and Owl Carousel JS
